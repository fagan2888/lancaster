Welcome to lancaster's documentation!
=====================================

A python extension wrapper for `avro-c`_.

Currently only supports reading a stream of avro serialized data.
Does not support writing, nor the avro container format.  For more
details on what lancaster is willing to read, see the
:doc:`spec`.

See also `the Avro project page`_.

.. _`the Avro project page`: https://avro.apache.org/
.. _`avro-c`: http://avro.apache.org/docs/1.7.7/api/c/index.html
